# ShopVista E-commerce

A beautiful, fully functional modern e-commerce website built with **HTML5**, **Tailwind CSS** (via CDN), and **vanilla JavaScript**.

**Live Demo:** Simply open `index.html` in any modern browser (or enable GitHub Pages for hosted version).

## ✨ Features

- **Elegant & Responsive Design** — Mobile-first premium UI with smooth interactions and delightful micro-animations
- **Product Catalog** — 12 curated premium products across 6 categories with high-quality imagery
- **Advanced Filtering & Search** — Real-time search, category filters, price range, and multiple sort options
- **Quick View Modal** — Detailed product view with dynamic size/color selection and quantity controls
- **Shopping Cart Drawer** — Persistent cart (localStorage) with live quantity adjustment, variant support, and totals
- **Seamless Checkout Experience** — Beautiful multi-step checkout modal with fake payment processing and order confirmation
- **User Session Simulation** — Login modal with persistent user state
- **Wishlist Functionality** — Heart toggle on product cards (demo)
- **Toast Notifications** — Elegant feedback for add to cart and other actions
- **Fully Client-Side** — No backend required. Works great offline after load.

## 🛠 Tech Stack

- Tailwind CSS Play CDN
- Vanilla JavaScript (ES6+)
- Font Awesome 6 icons
- Responsive grid + modern CSS animations
- LocalStorage for cart persistence

## 🚀 Getting Started

### View Locally
1. Clone or download this repository
2. Open the `index.html` file directly in your browser
3. Enjoy the fully interactive premium shopping experience!

### Deploy to GitHub Pages (Free Hosting)
1. Go to repository **Settings** → **Pages**
2. Under "Build and deployment", select **Source**: Deploy from a branch
3. Choose branch: `main` and folder: `/ (root)`
4. Click **Save** — your site will be live at `https://dasucosmos-eng.github.io/ecommerce-website`
